# Project Registry

Use this as the workspace-level registry for active projects.

## Status Meanings

- **Linked**: project has local instructions, GitHub repo, and Linear project.
- **Needs Linear**: project has local instructions and/or a GitHub remote, but no Linear project URL is recorded.
- **Needs GitHub**: project has local instructions, but no GitHub remote is present.
- **Needs decision**: decide whether this folder should become managed, archived, or local-only.
