# Aster Branding Resource

Use Aster as the default visual system for new app and frontend projects created with Codex Project OS.

Files:

- `BRAND.md` — agent-facing rules for layout, typography, color, components, and motion.
- `aster-tokens.css` — CSS variables and utility classes to copy or import into app code.

For a new app, read `BRAND.md` before UI work and copy or import `aster-tokens.css` into the app's style layer. Use a project-specific brand only when the user explicitly provides one.
